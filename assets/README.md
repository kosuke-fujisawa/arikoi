# assets/

## placeholder ファイルについて

`bg/placeholder_gate.png` と `bgm/placeholder_morning.ogg` は、tsumugai の `missing-asset` 検査（背景/BGMのパス実在チェック）を通すための空のダミーファイルです。中身は空で、実際の素材ではありません。

実際のフリー素材は issue #3（`assets: MVP用フリー素材を選定し assets-list.md でライセンスを管理する`）で選定・配置し、ライセンス情報は `docs/production/assets-list.md` に記録します。実素材が揃い次第、このディレクトリの placeholder ファイルは実素材に置き換えてください。
