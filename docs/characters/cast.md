# 現行キャラクター設定

本書を作品版の現行キャラクター正本とする。転換前の8ヒロイン構想とKaede単独短編の設定は参考資料であり、本書を拘束しない。

## 主人公

- 区分: 魔法大学の新入生
- 名前: 未定
- 特別な才能を持つわけではない
- 人の話を最後まで聞き、「どうして？」と問いかけることができる
- 誰かを論破するのではなく、理解しようとする
- 各ヒロインやミユキとの交流を通して、自身も成長していく

## 攻略ヒロイン

### レイカ

- 属性: 強気なお嬢様
- 学年: 同級生
- 役割: 対立と成長

名家の出身で、自信家。能力もプライドも高く、主人公にも遠慮なく物を言う。努力を惜しまず、負けず嫌いで、自分にも他人にも厳しい。実は責任感が非常に強い。

### ユズキ

- 属性: ほんわか系、料理上手
- 学年: 同級生
- 役割: 安心感と日常

優しく穏やかで、困っている人を放っておけない。料理が得意で学内でも評判。他人を優先しすぎ、自分の夢より周囲を優先する傾向がある。主人公との交流を通じて「自分自身の幸せ」について考え始める。

### アヤカ

- 属性: かっこいい先輩
- 学年: 一学年上
- 役割: 憧れと包容力

女性から非常に人気があり、頼りになって面倒見が良い。周囲から理想像を押し付けられ続けており、主人公の前だけは自然体になれる。

### イロハ

- 属性: 和風少女、武士道少女
- 学年: 同級生
- 役割: 信念と誠実さ

語尾に「ござる」を付ける。武士道と時代劇を好み、曲がったことが嫌い。非常に誠実で正義感が強く、自分にも厳しい一方、少し融通が利かない。

### サキ

- 属性: 余裕のある万能先輩
- 学年: 一学年上
- 役割: 余裕と導き

勉強も運動も優秀で、人を手のひらで転がす余裕があり、主人公をからかう。何でもできるからこそ、何かに本気になる経験が少ない。主人公にだけ興味を持つ理由は個別ルートで明かす。

### リゼ

- 属性: 無口、努力家
- 学年: 同級生
- 役割: 努力と将来

あまり笑わず、必要最低限しか話さない。真面目で、茶化されることを嫌う。明確な夢を持ち、そこへ向けて努力し続けている。恋愛より夢を優先していたが、主人公と出会い「一緒に歩む未来」について考え始める。

## 非攻略キャラクター

### ミユキ

- 属性: 趣味友達、距離感が近い、少しあざとい
- 学年: 同級生
- 攻略: 不可
- 役割: 恋愛ではない友情、主人公の日常、攻略ヒロインへの相談役

主人公と趣味がよく合い、距離感が近く、誰とでも仲良く話す。共通ルートでは主人公と一緒にいる時間が最も長いため、プレイヤーからは攻略できそうに見える。

主人公とは最後まで友達であり、恋愛関係にはならない。本人にも明確な理由はなく、「付き合うとか、なんか違う気がする」と感じている。恋愛や主人公を否定しているのではなく、ただそういう気分にならない。主人公はその感覚を尊重する。

個別ルートへ入ると出番は少しずつ減るが、主人公との友情は最後まで変わらない。「攻略できないヒロイン」ではなく、「恋愛という枠組みに入らない友人」として描く。
