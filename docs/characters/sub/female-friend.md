# 女友達（非攻略）

## 役割

非常に魅力的なキャラクターだが、最後まで攻略不可。

主人公とは完全に友人関係のまま終わる。

## 設計の意図

「攻略できなさ」が逆に作品のリアリティを強化する。

ギャルゲーでは全員が攻略可能であることが暗黙の前提だが、本作はその前提を一箇所だけ崩す。魅力的な女性キャラクターが「友人のまま」終わることで、プレイヤーに「現実の人間関係」を想起させる。

## 結末

男友達と付き合っていることが最終的に示唆される。

→ 関連: [male-friend.md](male-friend.md)

## 作品全体における機能

「ありふれた恋のカタチ」というタイトルの含意を広げる。恋愛にならない関係性もまた、リアルな「カタチ」のひとつである。
