# テーマ・作品の本質

## 実験としての作品

この作品は、

> 「特別な素材がなくても、人は誰かを好きになれるのか」

という問いの実験である。

そして同時に、

> 「理解されることは、孤独を壊せるのか」

という物語でもある。

桜、制服、放課後。どこにでもある風景。

だからこそ、その中に宿る感情だけは本物にしたい。

---

## 二層構造

本作は二つの問いを同時に走らせる。

| レイヤー | 問い |
|---|---|
| メタ的問い | フリー素材だけでも感情移入できるか |
| 物語内の問い | 理解されることで孤独は壊れるか |

この二つが最終的に重なるとき、作品が成立する。

---

## 各ヒロインが体現するテーマ

| ヒロイン | 孤独の形 |
|---|---|
| ツンデレ | 甘えられないこと |
| 電波系（哲学少女） | 理解されないこと / 他人を知りすぎてしまうこと |
| 妹 | 家族であること（血縁の不確かさ） |
| 幼なじみ | （要詳細化） |
| 姉ポジ | （要詳細化） |
| 生徒会長 | 孤高であること |
| 委員長 | 不器用であること |
| 元気系 | （要詳細化） |

---

## 解決の作法

主人公はヒロインの問題を「解決」しない。

問い続けること、側にいること、理解しようとすること。それだけで物語は動く。

電波系ルートにおいて主人公は「媒介者」として機能する。
彼女の思想を文化祭でクラスへ届けることで、孤立していた彼女が初めてクラスと「接続」される。
