# 0007. 舞台を魔法大学へ変更し、6人の攻略ヒロインと非攻略の友人を現行キャストとする

- 日付: 2026-07-21
- 状態: 採用

## 背景

これまでの現行英語版は、現代日本の高校を舞台に、主人公と幼馴染Kaedeの二人だけを描く短編として技術スパイクを行っていた。一方、作品版v0.1へ進むにあたり、旧8ヒロイン構想とKaede単独短編のどちらも採用せず、舞台と人物配置を抜本的に再構成することになった。

新構成では、恋愛だけでなく、対立、安心、憧れ、信念、導き、将来、異性間の友情をそれぞれ別の人物が担う。人物ごとの役割を重ねず、主人公の「論破せず、理解しようと問い続ける」性質を各関係から異なる形で描く。

## 選択肢

1. **Kaede一人の短編を完成させる**
   現行実装を最大限再利用できるが、作品全体の人物関係と複数ルートを表現できない。
2. **転換前の8ヒロイン構成へ戻す**
   既存の参考資料を使えるが、役割の重複があり、現在作りたい人物像とも一致しない。
3. **魔法大学を舞台に、6攻略ヒロインと非攻略の友人を置く（採用案）**
   各人物の役割を明確に分け、恋愛以外の関係も作品テーマへ含める。

## 採用案

- 舞台を現代日本の高校から、魔法大学へ変更する
- 主人公は魔法大学へ入学した新入生とする。特別な才能ではなく、人の話を最後まで聞き、「どうして？」と問いかけ、相手を理解しようとする姿勢を中心特性とする
- 攻略ヒロインは次の6人とする
  - レイカ: 強気なお嬢様。対立と成長
  - ユズキ: ほんわかした料理上手。安心感と日常
  - アヤカ: かっこいい先輩。憧れと包容力
  - イロハ: 武士道を好む和風少女。信念と誠実さ
  - サキ: 余裕のある万能先輩。余裕と導き
  - リゼ: 無口な努力家。努力と将来
- ミユキは攻略対象にしない。主人公と趣味が合い、距離が近い日常の友人であり、最後まで恋愛関係にならない
- ミユキと主人公が恋愛にならない理由を、秘密、別の恋愛、悲劇などで説明しない。「付き合うのは何か違う」という本人の感覚を主人公が尊重する
- 旧8ヒロイン資料とKaede設定は参考資料へ降格し、現行キャストを拘束しない

## 採用理由

- 7人の役割が明確に異なり、共通ルートと個別ルートで同じ機能を持つ人物が重複しにくい
- 主人公の「理解しようとする姿勢」を、異なる価値観や課題に対して繰り返し試せる
- ミユキによって、恋愛対象にならない異性間の友情を肯定的に描ける
- 魔法大学という舞台により、大学生活の自立性とファンタジー上の出来事を恋愛ルートへ結びつけられる

## 欠点・リスク

- 現行のKaedeプロローグ、シナリオE2E、素材計画は新設定と一致しなくなり、差し替えが必要になる
- 攻略対象が6人いるため、Kaede一人の短編よりシナリオ・素材・テストの規模が大きい
- 魔法大学の制度、魔法の範囲、生活様式はまだ定義されておらず、先に世界設定を増やしすぎる危険がある
- 「ござる」口調や強い属性表現が、英語版で同じ印象を保てるか検討が必要になる
- ミユキがプレイヤーに攻略対象だと誤認される設計は意図的だが、不満ではなく友情の肯定として受け取れる描写が必要になる

## 再評価条件

- v0.1で6ルートを同時に完成させることが現実的でないと判明した
- 魔法大学という舞台が、作品の中心である人物理解や「ありふれた恋」を弱める
- プレイテストで人物の役割が重複して見える
- ミユキの非攻略性が友情の肯定ではなく、単なる攻略制限として受け取られる

再評価時も、キャラクターを一括削除せず、最初に完成させる縦切りとリリース範囲を調整する。
