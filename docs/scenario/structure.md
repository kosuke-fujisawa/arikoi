# シナリオ構造

## 全体マップ

```
春（共通ルート前半）
  ↓
夏（共通ルート後半）
  ↓
秋（文化祭）← ここで分岐
  ↓
冬（個別ルート × 各ヒロイン）
```

## 各章

| 章 | 内容 | ファイル |

| 春 | 新学期・出会い | [spring.md](spring.md) |
| 夏 | 日常イベント・関係深化 | [summer.md](summer.md) |

---

## 分岐設計

秋の文化祭「誰と文化祭を回るか」によってルートが確定する。

個別ルートは冬に集約。各ヒロインの孤独や背景が掘り下げられ、最終的にはハッピーエンドへ到達する。多少ご都合主義でも、幸福で締める。

---

## エンディング方針

- 全ルート：ハッピーエンド
- バッドエンドは設けない（または最小限）
- 「ありふれた幸福」を誠実に描く
