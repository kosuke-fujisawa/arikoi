# 秋

## 概要

文化祭。「誰と文化祭を回るか」でルートが分岐する。

## 分岐点

文化祭の「誰と回るか」という選択が、個別ルートへの入口となる。

| 選択 | ルート |
|---|---|
| 幼なじみ | 幼なじみルート |
| 姉ポジ | 姉ポジルート |
| 生徒会長 | 生徒会長ルート |
| 委員長 | 委員長ルート |
| 元気系 | 元気系ルート |
| 電波系 | 電波系ルート |
| 妹 | 妹ルート |
| ツンデレ | ツンデレルート |

## 電波系ルートにおける文化祭の特別な意味

電波系ルートでは、文化祭そのものが物語のクライマックスとなる。

主人公が彼女の哲学・思想をクラスへ発表（媒介）し、それによって孤立していた彼女が初めてクラスと「接続」される。

## 演出上のポイント

- 文化祭の定番演出（クラス劇・模擬店・告白スポット）を網羅する
- 分岐前の共通パートで全ヒロインへの感情が最高潮に達するよう設計する
