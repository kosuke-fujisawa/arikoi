# 素材方針

## 基本方針

完全フリー素材のみを使用する。追加発注なし。

制約そのものを作品の前提にする。

## 使用素材カテゴリ

| カテゴリ | 方針 |
|---|---|
| 背景 | フリー背景素材を使用 |
| 立ち絵 | フリー立ち絵素材を使用 |
| BGM | フリーBGMを使用 |
| SE | フリーSEを使用 |
| フォント | フリーフォントを使用 |

## 素材の「見え方」について

フリー素材は「どこかで見たことがある」という感覚を生む。

本作ではこれを欠点ではなく、テーマの一部として扱う。「ありふれた素材」の中に「本物の感情」を宿らせることが、作品の実験そのものである。

## 作品性の成立のさせ方

素材の質ではなく、以下で作品性を作る。

- **見せ方**：どのタイミングでどの立ち絵を出すか
- **演出**：フェードイン・アウト・テキスト速度・SE配置
- **構成**：どんな順序でシーンを積み上げるか
- **テキスト**：会話・モノローグの質
