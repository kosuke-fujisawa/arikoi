# 演出方針

## 基本方針

徹底して平成ギャルゲーの文法を守る。

メタ的なコメントや自己言及はしない。「よくあるゲーム」として真剣に作る。

## 使うべき「お約束」

積極的に採用する定番要素：

| シーン | 要素 |
|---|---|
| 春 | 桜並木・制服・出会い |
| 夏 | 体育祭・夏祭り・部活 |
| 秋 | 文化祭・告白 |
| 冬 | 雪・制服・ハッピーエンド |
| 通年 | 放課後・ファミレス・買い食い |

## 演出の核心

「お約束を使う」ことと「誠実に描く」ことを両立させる。

定番イベントだからこそ、その中の感情を丁寧に積み上げる。プレイヤーは「知っているシーン」に安心しながら、「知らなかった感情」に気づく。

## タイトル表記について

「かたち」ではなく「カタチ」。

カタカナ表記にすることで2000年代ギャルゲーらしい空気感を演出する。凡庸で、ありふれていて、少しだけ懐かしいタイトルを目指す。

## UI方針

2000年代平成ギャルゲーのUIを踏襲する。

- メッセージウィンドウの位置・デザイン
- 選択肢の見せ方
- セーブ・ロード画面

「古さ」をネガとして扱わず、むしろ「懐かしさ」として機能させる。
